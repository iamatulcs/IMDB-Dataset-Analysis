# 🎬 IMDb Movie Data Analysis

## 📌 Project Overview
This project focuses on analyzing IMDb movie data using Python to uncover insights about movie ratings, genres, directors, popularity, and trends.

The analysis includes:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Statistical insights
- Correlation analysis
- Visualizations using Python libraries

The goal of this project is to practice real-world data analysis techniques and generate meaningful insights from movie datasets.

---

# 📂 Dataset Information
The dataset contains information related to movies such as:

- Movie Title
- Genre
- Rating
- Duration
- Director
- Votes / Rating Count
- Year of Release
- Gross Collection

# 📊 Key Analysis Performed

## ✅ Data Cleaning

## ✅ Exploratory Data Analysis
- Most common movie genres
- Highest rated movies
- Directors with most movies

# 🔍 Sample Insights

- Certain genres receive higher average ratings.
- Some directors consistently produce highly rated movies.
- Movie duration has a weak positive correlation with ratings.
- Popular movies generally have higher vote counts.

# ▶️ How to Run the Project


```bash

```

3. Install required libraries

```bash

```

4. Run the Jupyter Notebook

